# Employee Management Dashboard

## Project Overview
This is a React.js based Employee Management Dashboard.
It includes login authentication, employee CRUD operations,
search & filter, image upload, and print functionality.

## Tech Stack
- React.js
- React Router
- Context API
- Material UI / MDB UI
- Local Storage / Mock API
- bootstrap

## Features
- Login & Protected Dashboard
- Add / Edit / Delete Employee
- Image Upload with Preview
- Search & Filter (Gender, Status) or Name
- Print Employee List

## Steps to Run Project
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open browser at `http://localhost:5173`

## Assumptions / Design Decisions
- Authentication is mock based
- Data is stored in local state / local storage
- Single page application using React Router
